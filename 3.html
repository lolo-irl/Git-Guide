Comandos do Git


Quais são os principais comandos do Git que são usados com frequência?


git init: Inicializa um repositório Git em um diretório local.
git clone: Clona um repositório remoto para o local.
git add : Adiciona arquivos específicos ao staging.
Use git add: para adicionar todos os arquivos.
git commit -m "mensagem": Cria um commit com as mudanças no staging, salvando-as no histórico.
git push: Envia os commits locais para o repositório remoto.
git pull: Atualiza o repositório local com as mudanças do repositório remoto.
git status: Mostra o status dos arquivos no repositório local.
git log: Exibe o histórico de commits.
git checkout : Muda para uma branch diferente.
git merge : Mescla outra branch na branch atual.
